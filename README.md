Repositório com exercícios e projetos desenvolvidos em diferentes cursos de Python, abrangendo desde fundamentos da linguagem até tópicos mais avançados como análise de dados e automação de processos.

## 🎯 Sobre

Este repositório reúne minha trajetória de estudos em Python, organizada por curso/tema. A ideia é documentar a prática e evolução ao longo do aprendizado, servindo tanto como portfólio quanto como material de consulta pessoal.

## 📂 Estrutura do repositório

```
Estudando_Python/
├── Python_para_dados/     # Exercícios voltados para análise e manipulação de dados
├── (adicionar outras pastas conforme novos cursos)
└── README.md
```

## 🛠️ Tecnologias e tópicos abordados

- **Fundamentos de Python** — sintaxe, estruturas de controle, funções, POO
- **Python para Dados** — bibliotecas como Pandas, NumPy e manipulação de datasets
- **Automação de Processos** — scripts para tarefas repetitivas

## 🚀 Como usar

```bash
git clone https://github.com/Nic-Rocha/Estudando_Python.git
cd Estudando_Python
```

Cada pasta contém os exercícios referentes ao curso/tema correspondente, com comentários explicando a lógica de cada solução.

Repositório em constante atualização conforme avanço nos estudos de Ciência da Computação e Ciência de Dados.
